# TB5 HTML Projects

This repository contains multiple frontend mini-projects built with plain HTML/CSS/JS plus Bootstrap, Tailwind, and jQuery.

## Projects Overview

### `mt` - Inventory Management Variants
Different implementations of an inventory manager app (add/edit/delete products, search/filter, details view, local data handling).

- `mt/Claude-Code/index.html`
  - Tailwind + jQuery inventory UI with stats, filters, pagination, and details panel.
- `mt/GitHub-Copilot/index.html`
  - Tailwind + jQuery inventory app in a dashboard/table layout with modal interactions.
- `mt/Google-Antigravity/Bootstrap-jQuery/`
  - `index.html`, `style.css`, `app.js`
  - Bootstrap + jQuery inventory system with product modals and styled components.
- `mt/Google-Antigravity/Tailwind-jQuery/`
  - `index.html`, `app.js`
  - Tailwind + jQuery inventory version with a modern glassmorphism style.

### `mth` - AutoCare Manage
Car service record management app.

- `mth/index.html` + `mth/style.css`
  - Bootstrap + jQuery version.
  - Features: add/update records, validation, search, alerts, and `localStorage` persistence.
- `mth/tailwind_index.html`
  - Tailwind-based UI version of the same AutoCare concept.

### `ssdl` - StayFinder Rental Platform
Rental listing concept app for renters and owners.

- `ssdl/StayFinder.html`
  - Tailwind-style page for posting and browsing rental listings.
- `ssdl/StayFinderBootstrap.html`
  - Bootstrap version with hero search, filters, listing cards, owner form, and stats.
- `ssdl/StayFinder1.jpg`, `ssdl/StayFinder2.jpg`
  - Preview/mockup images.

### `tzh` - VelvetBrew POS
Coffee shop POS (Point of Sale) and analytics dashboard.

- `tzh/index.html`
  - Tailwind + jQuery single-page app.
  - Features: menu browsing, cart/order flow, payment selection, sales history, and dashboard metrics using `localStorage`.

## How To Run

These are static frontend projects, so no build step is required.

1. Open any `index.html` (or the named HTML file) directly in a browser.
2. For best development workflow, use a local static server (for example VS Code Live Server).

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla + jQuery)
- Bootstrap 5
- Tailwind CSS (CDN)
